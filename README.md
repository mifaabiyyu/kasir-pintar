## Requirements

1. PHP v8
2. NPM v9.5.1
3. Node v18.16.0
4. Composer

## Running Project

1.  Clone Repository.
2.  Create .env file and copy all .env-example file to .env
3.  Create MySQL database with name "kasir_pintar".
4.  run "composer install -w --ignore-platform-reqs"
5.  run "php artisan migrate"
6.  run "php artisandb:seed"
7.  Open projects
